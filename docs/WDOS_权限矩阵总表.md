## 一、角色定义

| 角色 | 标识 | 说明 |
|------|------|------|
| 一线人员 | handler | 接单、处理工单的一线员工 |
| 领班 | supervisor | 班组负责人，管理同部门一线 |
| 经理 | manager | 部门负责人 |
| 总监 | director | 跨部门管理者 |
| 管理员 | admin | 系统管理员 |
