## 二、权限配置页对应关系

管理员在「权限配置」页面为每个角色勾选的权限，与上表的对应关系：

```
权限配置页模块          →  影响上表中的哪些行
─────────────────────────────────────────────
工单模板管理           →  工单模板全部权限
工单数据管理           →  工单数据全部权限（含查看范围限制）
抑制策略配置           →  抑制策略全部权限
区域路由配置           →  区域路由全部权限
SLA 策略配置           →  SLA 策略全部权限
用户管理               →  用户管理全部权限
部门管理               →  部门管理全部权限
排班管理               →  排班全部权限（查看 + 编辑 + 导入）
排班导入               →  排班导入权限（独立开关）
排班修改               →  排班修改权限（独立开关）← 你特别要求的
统计报表               →  统计报表全部权限
```
